Frequently Asked Questions

How do I get started with PointIQ?

Simply download the app from the App Store, create an account, and start tracking your matches! The app is designed to be intuitive and fast to use during real matches.

How does cloud sync work?

PointIQ uses Supabase for secure cloud storage. Your match data is automatically synced when you're online, and stored locally for offline access. All data is encrypted and protected with Row Level Security (RLS).

Can I use the app offline?

Yes! PointIQ stores data locally and will sync to the cloud when you're back online. You can track matches and log strokes even without an internet connection.

What stroke types can I log?

The app supports a comprehensive taxonomy of strokes including loops, pushes, blocks, chops, and more. You can log strokes during or after rallies for detailed analysis.

Is my data private and secure?

Absolutely. PointIQ collects only the data required for core functionality. All data is protected with Row Level Security, and users can only access their own matches and logs. No sensitive secrets are stored in the client.

How do I view my match history and statistics?

Navigate to the History tab in the app to review past matches, stroke distributions, and performance trends. The app provides detailed analytics to help you improve your game.

Troubleshooting

I'm seeing "Supabase not configured" warning
  • Make sure you've updated Supabase credentials in the app settings
  • Verify the URL and key are correct (no extra spaces)
  • Check your internet connection
Data is not syncing to the cloud
  • Check your internet connection
  • Verify your Supabase credentials are correct
  • Ensure you're signed in to your account
  • Data will sync automatically when connection is restored
The app crashes or behaves unexpectedly
  • Make sure you're running the latest version of the app
  • Try force-closing and reopening the app
  • If the issue persists, contact support with details about what you were doing when it occurred
I can't log strokes during a match
  • Ensure you're in an active match
  • Check that the match hasn't ended
  • Try restarting the match if needed

Privacy & Data

PointIQ collects only the data required to provide its core functionality:

  • User account information
  • Match and stroke data created by the user

All data is stored securely with Row Level Security (RLS) policies. Users can only access their own data. For full details, please refer to the app's Privacy Policy.
